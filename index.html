In this project we publish as open data a list of pharmacies which sell the morning-after pill in the city of Catania. These pharmacies has been tested by the Rebeldesse group, which operates in the scope of Officina Rebelde.

Open Data

The municipality of Catania provides a dataset Farmacie which contains all pharmacies in the city.
